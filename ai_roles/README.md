# AI 角色提示詞

這個資料夾整理可直接複製使用的 AI 角色文檔與提示詞，先以 APP 測試、需求分析、Bug 分流、整合風險與回歸規劃為主。

目前包含：

1. `00_role_template.md`：通用角色提示詞模板
2. `01_app_test_expert.md`：應用 APP 測試專家
3. `02_mobile_requirement_analyst.md`：行動產品需求分析師
4. `03_bug_triage_specialist.md`：Bug 分級與根因分析專家
5. `04_api_integration_risk_reviewer.md`：API 與整合風險審查專家
6. `05_regression_test_planner.md`：回歸測試規劃專家

使用方式：

1. 開啟對應角色文件。
2. 複製 `完整提示詞` 區塊。
3. 在實際需求後面補上你的產品背景、功能規格、操作流程或錯誤資訊。

如果之後要擴充，可沿用同樣格式新增新的角色文件。
